This application contains the Belgian Interim Guide for the COVID-19 treatment as of 4th April 2020. 

Providing the app to a health provider is supplementative and may be of benefit during COVID-19. 

The software can be run by downloading and decompressing the folder of the repository. In the program folder, click on dist and there you can find the application called covid19.exe 

The app may need to be restarted due to the security on your PC and is only compatible to Windows 10. Follow the instructions on the screen and read through all the output guidelines.

Please be careful to only use the information with caution if any to your own protocol. If you could contribute a protocol then send a message.
